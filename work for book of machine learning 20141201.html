機械学習の勉強

Permalink: 2014-12-01 15:34:00+09:00 by ishikawakouji in Machine Learning tags: machine learning Building Machine Learning System with Python 実践 機械学習システム

理論は難しいのでパス

Wikipediaを見ると、 Andrew Ng先生の名前がありました。

scikit-learnではサポートされていないので、 gensimパッケージを利用するようです。 pipなどでインストールできるようです。

pip install gensim

この本では、文章をトピックに分類する例題です。

文章から単語を抜き出し、その単語がどのトピックに相当するかを識別し、 単語を含んでいた文章がどのトピックに相当するかを推測するというものです。

トピックモデルは中間的な役割にも利用できる

トピックで比較をすると、同じトピックについて論じられている文書は似ていると言えるでしょう。 しかし、単語を直接比較するのとはちょっと訳が違うということになります。同じ事項を、別の表現で 言うことがある(首相とか、安倍さんとか)ので、これをうまくまとめられるようです。

また、単語を直接比較するより、トピックで比較した方が個数を減らせるので、 次元の削減という意味もあります。
