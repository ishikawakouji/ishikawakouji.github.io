サンプルファイルの場所

Pylearn2 を GitHub から持ってきている場合は、ここにある。

pylearn2/scripts/tutorials/softmax_regression

regression と書いてあるが回帰ではない

本文中にも誤った名前だと書いてある。

学習アルゴリズムを設定して実行する

パラメータとか

難しいことは本文にまかせて、動作の内容やパラメータの意味を示す。 本文にも書いてあるが、一度日本語にしておいたほうがよさそう。

  • pylearn2のBGDクラスを利用する。batchでの勾配降下法。
  • batch_size はBGDを適用する間隔
  • line_search_mode は再急降下法になる
  • conjugateは、boolearnで、1にすると、非線形共役勾配降下法になる

モニタリング

観察する量を channel に設定して、 monitoring すればいい。

BGDでの終了条件

モニタベースであれば、 criterion を利用できる。 MonitorBased は他にも引数を持つが、気にせず default を使えばいい。

MNIST のデータをとりあえず利用する

データをロードするしくみを理解していないので、ありモノを利用する。 そのために、MNISTのデータを一式持ってきておく。
