JDK8をUbuntu系にインストールする

Permalink: 2014-12-11 17:49:00+09:00 by ishikawakouji in setup tags: Linux Ubuntu lubuntu Java JDK

マニュアルでインストール

普通にインストールすると openjdk がインストールされるので、 OracleのJDKをインストールするのであれば、手でインストール

参照先：Debian系LinuxへOracle JDKをインストールする

JDK8をダウンロード

Oracle JDK からダウンロードする。

展開して移動

$ tar zxvf jdk-8u25-linux-x64.tar.gz
$ sudo mkdir /usr/lib/jvm
$ sudo mv ./jdk1.8.0_25 /usr/lib/jvm/jdk1.8.0

セットアップ

まず、登録。

$ sudo update-alternatives --install "/usr/bin/java" "java" "/usr/lib/jvm/jdk1.8.0/bin/java" 1
$ sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/lib/jvm/jdk1.8.0/bin/javac" 1
$ sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/lib/jvm/jdk1.8.0/bin/javaws" 1

次に選択。あらかじめ java は抜いているので、競合はない。

$ sudo update-alternatives --config java
リンクグループ java に 1 つの alternative のみがあります (/usr/bin/java が提供): /usr/lib/jvm/jdk1.8.0/bin/java
設定は行いません。

バージョンを確認。

$ java -version
java version "1.8.0_25"
Java(TM) SE Runtime Environment (build 1.8.0_25-b17)
Java HotSpot(TM) 64-Bit Server VM (build 25.25-b02, mixed mode)

おけ。 さらに、ダメ押し？

sudo update-alternatives --config javac
sudo update-alternatives --config javaws
